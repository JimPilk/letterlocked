Tutorial

Each cipher has a rough difficulty and in order to beat each level, you must decode a piece of ciphertext from that level! The ciphertext resets every day, so it's a new puzzle each day! Feel free to do these in any order you want, and using any tools you want from a pen and paper to a full programmed solution. I recommend you research each cipher before attempting to crack it (wikepedia is very useful for this), as the descriptions are more a basic outline with some historical context and fun facts mixed in
